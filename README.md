# Papamoa Tourism Website

A responsive tourism website showcasing Papamoa, New Zealand.

## Features

- Responsive HTML/CSS design
- Hero video
- Local attractions
- Modern navigation
- Multimedia content
- Designed as a portfolio project

## Technologies

- HTML5
- CSS3
- JavaScript

## Status
