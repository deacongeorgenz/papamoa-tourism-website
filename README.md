# Papamoa Tourism Website

A personal front-end web development project built to explore modern website layout techniques using HTML, CSS and JavaScript.

This project was not created as an assessment. I built it as a self-directed learning project to practise hero sections, card layouts, responsive design, multimedia content, and basic JavaScript features through a tourism-style website for Pāpāmoa, New Zealand.

## Project Purpose

The goal of this project was to improve my front-end development skills by building a realistic multi-page tourism website with:

- A video hero section
- Navigation layout
- Attraction cards
- Local imagery
- Responsive design
- Weather and surf information
- A clean tourism-focused user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- Tabler Icons
- Open-Meteo API
- Open-Meteo Marine API

## Features

- Landing page with video hero background
- Attractions page with card-based layout
- Live weather and surf conditions widget
- Mobile-friendly video support
- Clean visual layout for tourism content
- Local Pāpāmoa theme and imagery

## What I Learned

Through this project I practised:

- Building modern hero sections
- Creating reusable card-style layouts
- Structuring a multi-page website
- Working with media assets
- Using JavaScript to fetch live API data
- Improving layout consistency with CSS
- Designing for both desktop and mobile users
